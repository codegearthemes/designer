Designer plugin will add custom made elementor elements to elementor page builder to customize sites with ease.

== Description ==

Designer is a powerful and advanced all in one Elementor addons that help you to create a beautiful website with ease. It offers many customizations and unique styling options to enhance the look and feel of every element on your website.

Build a beautiful website with Designer in less time without any coding skills.

## 42+ Elementor Widgets

= Countdown Widget =
Create excitement on your site with Elementor's Countdown widget. Easily showcase upcoming events or product launches with this customizable, user-friendly feature. Add a sleek and dynamic countdown to your Elementor-powered pages for a visually engaging and time-sensitive touch.

= Video Popup =
Enhance your website's interactivity with Elementor's Video Popup widget. Easily embed and customize video pop-ups for dynamic content display. Perfect for showcasing promotional clips or tutorials, this widget adds a visually captivating and user-friendly touch to your site. Elevate engagement effortlessly with this versatile and seamless integration into your Elementor-powered pages.

= Template =
Optimize your site's appearance with Elementor's Template Showcase widget. Customizable and responsive, it effortlessly displays your saved templates, making it perfect for highlighting diverse content like portfolios, products, or services. Elevate your website with visually striking and user-friendly template presentations, seamlessly integrating with Elementor for a seamless user experience.

= Pricing table =
Elementor's Pricing Table widget streamlines pricing presentation, offering customization and responsiveness. Ideal for showcasing products or services, it enhances websites with visually appealing and user-friendly pricing information.

= Image hotspots =
Elementor's Image Hotspots widget boosts engagement by adding clickable points to images, ideal for dynamic product showcases or interactive content.

= Slider =
Enhance your website with a sophisticated yet lightweight image slider featuring advanced functionalities and customizable settings. Elevate your user experience by seamlessly integrating this dynamic element into your site.

= Product slider =
Effortlessly showcase your products using the Products Slider Elementor widget. Customize the slider to match your brand and engage your audience with a dynamic presentation.

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the Designer plugin through the 'Plugins' menu in WordPress.

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Designer, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Designer" and click Search Plugins. Once you’ve found our  plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

== Frequently Asked Questions ==

= Does it work with any theme?

It will work with any theme as long as Elementor plugin is installed.

= Can I use the plugin without Elementor Page Builder?

No. You cannot use without Elementor since it’s an addon for Elementor.

= Will this plugin work with Elementor? =

This plugin adds custom made elementor widgets which helps to customize with elementor builder with ease.
